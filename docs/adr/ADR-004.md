# ADR-004: Two User Roles Only

**Status:** Accepted  
**Date:** 2026-06-28  
**Last Updated:** 2026-06-28

## Context

Minimize RLS complexity and scope; other roles can be added per-phase

## Decision

Limit MVP to Student and Organizer roles only

## Consequences

Student + Organizer minimizes RLS complexity; other roles added per-phase
