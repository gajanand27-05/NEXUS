# ADR-003: Generic Event Model

**Status:** Accepted  
**Date:** 2026-06-28  
**Last Updated:** 2026-06-28

## Context

Domain model must support future event types (workshops, contests, bootcamps)

## Decision

Use generic domain naming (events, applications) instead of hackathon-specific terms

## Consequences

Tables use events, applications not hackathons; supports workshops, contests, bootcamps without renaming
