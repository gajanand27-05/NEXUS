# ADR-005: No Custom Backend

**Status:** Accepted  
**Date:** 2026-06-28  
**Last Updated:** 2026-06-28

## Context

MVP does not need Express, FastAPI, or any additional server

## Decision

Use Server Actions + RLS instead of a custom backend server

## Consequences

Server Actions + RLS handle all MVP data flow; FastAPI only added when AI features require Python
