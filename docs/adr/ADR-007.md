# ADR-007: No External State Library

**Status:** Accepted  
**Date:** 2026-06-28  
**Last Updated:** 2026-06-28

## Context

Server Components + useState + Context cover all MVP needs

## Decision

Do not use Redux, Zustand, React Query, or any state management library in MVP

## Consequences

Server Components + useState + Context cover all MVP needs; React Query can be added later
