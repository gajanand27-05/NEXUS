# ADR-008: RLS for Authorization

**Status:** Accepted  
**Date:** 2026-06-28  
**Last Updated:** 2026-06-28

## Context

Security must be enforced at database level, not just application level

## Decision

Enforce all authorization through Supabase Row-Level Security policies

## Consequences

Security enforced at database level; cannot be bypassed by client code; single source of truth
